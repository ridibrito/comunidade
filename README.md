### App Structure

Key routes under `src/app/(app)`:
- `dashboard`, `catalog`, `courses/[slug]`, `watch/[lessonId]`
- `tracks/[slug]`, `books/[slug]`, `tools/[slug]`
- `events`, `community`, `profile`, `admin`

Marketing landing at `src/app/(marketing)/page.tsx`.

### Supabase

Client utilities at `src/lib/supabase.ts` using `@supabase/ssr`.

### AI (Mentora)

Minimal example using `OPENAI_API_KEY` in `src/lib/ai.ts` and server action `src/app/(app)/community/actions.ts`.

## Getting Started

First, run the development server:

```bash
npm run dev

# or

# or
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
